# NutriLens AI 🥗

NutriLens AI is a premium, AI-powered nutrition analysis platform built for the modern health-conscious individual. Snap a photo, get instant nutritional insights, and make smarter food choices with the power of Gemini AI.
Deployed Project link : https://nutrilens-ai-prod-v2-yzr4i5yoqa-uc.a.run.app

## 🚀 Features

- **Snap & Analyze**: Instant recognition of food items and portion sizes from images.
- **Nutritional Breakdown**: Detailed calories, macros (Protein, Carbs, Fat), and health scores.
- **AI Recommendations**: Personalized suggestions for healthier eating habits.
- **Meal Comparison**: Side-by-side analysis of two meal options to find the healthier choice.
- **Healthy Places**: Location-aware restaurant suggestions with healthy menu options.
- **Modern Dashboard**: Track your nutrition journey with beautiful, interactive charts.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **AI Engine**: Google Gemini 1.5 Flash
- **Charts**: Recharts
- **Components**: shadcn/ui inspired custom glassmorphism components

## 📦 Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**:
   Copy `.env.example` to `.env.local` and add your API keys:
   ```bash
   cp .env.example .env.local
   ```
4. **Run the development server**:
   ```bash
   npm run dev
   ```

## 🔑 API Keys Needed

- **Gemini API Key**: Get it from [Google AI Studio](https://aistudio.google.com/)
- **Google Maps/Places API Key**: Get it from [Google Cloud Console](https://console.cloud.google.com/)
